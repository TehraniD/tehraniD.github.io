Lab 4 Kinetics — Version 2: Scheme 1 Design

CHM152 · Bromination of Acetone · Revised 5-Run Design

INSTRUCTOR TEACHING GUIDE V2 — EASIER FOR STUDENTS

🔄 Why This Version Is Better for Students

The original 3-run design asks students to infer zero order in Br₂ by comparing R² values from three different linearization plots. This requires understanding WHY those plots differ — a hard conceptual leap when students are already overwhelmed by the rate law.

This version borrows from Scheme 1: vary [Br₂] directly while holding everything else constant. The result is visual and undeniable.

❌ Original Approach
  • 3 runs, only R² comparison reveals order
  • Students need to understand 3 different plot types
  • Zero order conclusion feels abstract
  • Easy to get confused about which graph "wins"
VS
✅ V2 Scheme 1 Approach
  • 5 runs, direct visual proof of zero order
  • ALL plots are [Br₂] vs time — one graph type only
  • "Same slope despite 4× more Br₂" is visceral
  • Students see it, not just calculate it
💡
The punchline students will experience: Run A uses 1 mL of Br₂ and turns colorless in ~3 min. Run C uses 4 mL of Br₂ — the solution is four times more orange — but the rate of color disappearance is identical. That moment teaches zero order better than any graph comparison.

📊 Expected Results — What Students Should See

Part 1: Runs A, B, C — The Zero Order Proof

Students plot [Br₂] vs time for all three runs on the same Excel graph. They add a trendline to each. The slopes should be equal:

Run[Br₂]₀Time to completion (est.)Slope = −k'What students observe
A1×10⁻³ M~2–3 min−k' (value X)Pale, goes colorless fast
B2×10⁻³ M~4–6 min−k' (≈ X)Orange, takes twice as long
C4×10⁻³ M~8–12 min−k' (≈ X)Deep orange, takes 4× as long
✅
The punchline: All three slopes are equal. The time to complete increases (more Br₂ = more to consume), but the rate of disappearance is constant. This is the definition of zero order. Students see three parallel lines starting at different heights.
What to say to students during this moment

"Look at your graph. Run A, B, and C — three different amounts of Br₂ — and the lines are parallel. The slopes are the same. Changing the amount of Br₂ changes how long it takes to finish, but not how fast it disappears per second. That's what zero order means. Br₂ is not involved in the slow step."

Part 2: Runs D and E — Finding n and m

Using Run B as baseline (slope = k'_B):

ComparisonWhat changedExpected slope ratioConclusion
D vs B[H⁺] doubled (0.20→0.40 M)slope_D / slope_B ≈ 2m = 1 (first order in H⁺)
E vs B[acetone] doubled (0.80→1.60 M)slope_E / slope_B ≈ 2n = 1 (first order in acetone)
rate = k [acetone]¹ [H⁺]¹ [Br₂]⁰
n = 1, m = 1, p = 0   |   rate = k [acetone][H⁺]

⏰
Run C is the time constraint. At 4×10⁻³ M Br₂ with p=0, it takes ~4× longer than Run A to go colorless. Start it promptly. If time is short, 8 minutes of data (going from 0.48 down to ~0.08 absorbance) is sufficient for a clean slope.
🔍
Run E will finish fastest. With double acetone, k' doubles → the reaction goes twice as fast as Run B. Students often think it will be slower because "more stuff." Watch for this — it's a great teaching moment about what the rate law actually means.

💻 Week 2 Excel Analysis — Simpler Than V1

🎯
Good news for students: They only need ONE type of graph — [Br₂] vs time — for ALL five runs. No choosing between zero/first/second order plots. The comparison does the work.

Step 1: Convert absorbance → [Br₂] for all 5 runs

For each run's sheet:
Column A: absorbance  |  Column B: time  |  Column C: =(A2 - intercept)/slope

Step 2: Plot ALL 5 runs on one graph (or two graphs)

Graph 1 (Part 1): Plot [Br₂] vs time for Runs A, B, C on the same axes.
Add separate trendlines to each. Record slope of each line.

Graph 2 (Part 2): Plot [Br₂] vs time for Runs B, D, E on the same axes.
Add trendlines. Compare slopes to Run B.

Step 3: Part 1 Analysis — Zero Order Proof

Students fill in:
slope_A = _____    slope_B = _____    slope_C = _____
"Are these slopes equal? If yes → p = 0 (zero order in Br₂)"
% difference = |slope_A − slope_B| / average × 100 = ____%

Step 4: Part 2 Analysis — Orders n and m

slope_D / slope_B = _____ ≈ 2ᵐ → m = _____
slope_E / slope_B = _____ ≈ 2ⁿ → n = _____

Step 5: Calculate k

k = k' / ([acetone]ⁿ × [H⁺]ᵐ)    using slope from each run as k'

Calculate from Runs B, D, and E separately (they all have same [Br₂] so k' is directly comparable). Average the three k values.

🧠 New Misconceptions This Version Surfaces

"Run C took much longer — so isn't it slower?"

Address it: No. The RATE (slope = −d[Br₂]/dt, M/s) is the same. But Run C started with 4× more Br₂ so it takes 4× as long to run out. Think of draining a bathtub — same drain rate, but a bigger tub takes longer to empty.

"The slopes look slightly different for A, B, C — does that mean Br₂ matters a little?"

Address it: Experimental variation will produce slightly different slopes. Calculate % difference. If within ~10–15%, the conclusion is zero order. No experiment is perfect — the question is whether the differences are within reasonable error.

"Run E is faster — but I added more acetone, so I expected more crowding and slower"

Address it: More acetone means faster enolization (more substrate for the rate-determining step), so faster rate. Concentration is fuel for the reaction, not crowding. More fuel → faster rate (when n>0).
