📝 Student Information
🔬 What Is a Buffer?
A buffer is a solution that resists pH changes when small amounts of acid or base are added. Think of it like a chemical sponge — it absorbs added H⁺ or OH⁻ ions without significantly changing the solution's pH.
How Does It Work?

A buffer has two components working as a team:

•   A     (HA): ready to neutralize any added base (OH⁻) ✓ weak acid

•   Its conjugate     (A⁻): ready to neutralize any added acid (H⁺) ✓ base

When acid is added:   A⁻ + H⁺ → HA   (conjugate base absorbs the acid)
When base is added:   HA + OH⁻ → A⁻ + H₂O   (weak acid absorbs the base)
Today's Buffer System

• Weak acid:     (CH₃COOH), Ka = 1.8 × 10⁻⁵ ✓ acetic acid

• Conjugate base: Sodium acetate (    / CH₃COO⁻) ✓ NaCH₃COO

Key Equation — Henderson-Hasselbalch
pH = pKa + log ([A⁻] / [HA])
⚠️ Important: This equation only works for true buffers (Beakers 2 and 3 — where both HA and A⁻ are present).
For pure acid (Beaker 1) or pure conjugate base (Beaker 4), you must use an ICE table.

Step 1: Calculate Concentrations Using Dilution
Stock solutions are both 0.5 M. When you mix them, concentrations change because you dilute into the total 10 mL volume.

General formulas:
[HA] = (Volume of acid used × 0.5 M) ÷ 10 mL
[A⁻] = (Volume of base used × 0.5 M) ÷ 10 mL

[HA] = (8 mL × 0.5 M) / 10 mL = 4.0 / 10 = 0.40 M
[A⁻] = (2 mL × 0.5 M) / 10 mL = 1.0 / 10 = 0.10 M

Substitute into Ka expression:

Ka = x² / (0.5 − x) ≈ x² / 0.5 = 1.8 × 10⁻⁵

x² = (1.8 × 10⁻⁵)(0.5) = 9.0 × 10⁻⁶

x = [H⁺] = M ✓ 3.0 × 10⁻³

pH = −log(3.0 × 10⁻³) = ✓ 2.52

Step 4: ICE Table for Beaker 4 (Pure  )
Beaker 4 has only sodium acetate (0.50 M). The acetate ion (CH₃COO⁻) acts as a weak base in water. We need Kb first.

First, calculate Kb:

Ka × Kb = Kw = 1.0 × 10⁻¹⁴

Kb = Kw / Ka = (1.0 × 10⁻¹⁴) / (1.8 × 10⁻⁵)

Kb = M ✓ 5.56 × 10⁻¹⁰
