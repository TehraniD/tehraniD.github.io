# CHM152L Lab 3: Heat of Fusion - Complete Interactive Package

This package includes an interactive lab guide and Canvas assessment quiz for the Heat of Fusion experiment.

## 📦 Package Contents

### 1. **Lab3_Interactive_HeatOfFusion_Complete.html**
   - Comprehensive interactive lab guide
   - All sections included (Learning Guide, Pre-Lab, Safety, Procedure, Data & Calculations, Analysis, Post-Lab)
   - Auto-calculating data tables
   - PDF generation capability
   - Auto-save feature

### 2. **Lab3_HeatOfFusion_Quiz_Canvas.zip**
   - Canvas-ready quiz in QTI 1.2 format
   - 20 questions, 100 points total (5 points each)
   - Mix of question types: Multiple Choice, Multiple Answer (Select All), True/False
   - Difficulty level 5-7

---

## 🚀 How to Use the Interactive Lab Guide

### For Students:

1. **Open the HTML file** in any modern web browser (Chrome, Firefox, Safari, Edge)
2. **Navigate through tabs** at the top to move between sections
3. **Complete sections in order:**
   - Start with "Learning Guide" to understand concepts
   - Complete "Pre-Lab" questions before the lab
   - Review "Safety" procedures and check all items
   - Follow step-by-step "Procedure" during the lab
   - Enter data in "Data & Calculations" - calculations happen automatically!
   - Review "Analysis" for your results and error analysis
   - Answer "Post-Lab" questions

4. **Save your work:**
   - Click "Save All Data to Browser" - your work is saved locally
   - Click "Load Saved Data" to resume later
   - Auto-save runs every 30 seconds

5. **Generate PDF report:**
   - Click "Print / Save as PDF"
   - In the print dialog, select "Save as PDF"
   - All your data, calculations, and answers will be included

### For Instructors:

#### **Customization Options:**
- Edit the HTML file to modify:
  - Learning guide content
  - Pre-lab and post-lab questions
  - Safety procedures
  - Procedure steps
  - Theoretical values

#### **Grading:**
- Students submit the PDF report generated from the interactive guide
- All calculations are shown in the PDF
- Pre-lab and post-lab answers are included
- Error analysis and sources of error are documented

8. **One Question at a Time:** Optional

### Step 4: Review and Publish

1. **Preview the quiz** to ensure all questions imported correctly
2. Check that all 20 questions are present

## ❓ Troubleshooting

### Interactive Lab Guide Issues:

**Q: Data isn't saving**
- A: Check that JavaScript is enabled and allow local storage in browser settings

**Q: Calculations aren't updating**
- A: Click the "Calculate All Values" button or refresh the page

**Q: PDF looks wrong**
- A: Use Chrome or Firefox for best PDF output. Try adjusting page scaling in print dialog.

**Q: Lost my data**
- A: If you saved to browser, click "Load Saved Data". If not saved, data cannot be recovered.

### Canvas Quiz Issues:

**Q: Questions didn't import**
- A: Ensure you're using the ZIP file, not individual XML files. Try importing to a question bank first.

**Q: Question formatting looks wrong**
- A: This is normal for complex equations. Preview each question and adjust if needed.

**Q: Wrong point values**
- A: Manually adjust point values in quiz settings after import.

## 📝 Answer Key for Quiz

### For Instructor Reference Only:

1. **B** - Energy required to convert solid to liquid at constant temperature
2. **True** - Temperature stays constant during phase change
3. **A, B, D** - Calorimetry assumptions (not C - final temp is NOT always average)
4. **C** - 6,680 J (20.0 g × 334 J/g)
5. **True** - Wet ice causes low ΔH_f
6. **B** - Heat loss makes ΔH_f too high
7. **A, B, D** - Hydrogen bonding explains high ΔH_f (not C - molecular weight)
8. **A** - Melting requires more energy (16,700 J vs 8,368 J)
9. **True** - Calorimeter minimizes heat exchange
10. **B** - 361.2 J/g (calculation: 120 × 4.184 × 18 / 25)
11. **A, B** - Heat loss and cold ice make ΔH_f high (wet ice makes it low)
12. **True** - Three-step calculation needed for ice below 0°C
13. **B** - Horizontal plateau at 0°C represents heat of fusion
14. **A, C, D** - Best practices (not B - add ice gradually, not all at once)
15. **B** - 10.3% error (|367.9 - 333.55| / 333.55 × 100)
16. **True** - Units differ because c involves ΔT
17. **B** - High heat of fusion keeps things cold
18. **A, B, C, D, E** - All five steps needed for complete phase transition
19. **B** - ΔH_f represents change in potential energy
20. **True** - Heat of vaporization > heat of fusion due to more bonds broken
