Lab 4: Rate of a Reaction — V2

Scheme 1 Design · CHM152 Student Interactive Guide

What You're Doing and Why

You're figuring out the rate law for acetone reacting with bromine. The rate law tells you which ingredients control how fast the reaction goes — and by how much.

rate = k [acetone]ⁿ [H⁺]ᵐ [Br₂]ᵖ

Your job: find the actual values of n, m, and p. The result will be surprising.

The 5-Run Plan

Part 1 — The Zero Order Proof (Runs A, B, C)

Three reactions. Same [HCl] and [acetone] in all three. Different amounts of Br₂: 1×, 2×, and 4×. You measure how fast Br₂ disappears in each. If the rate is the same in all three despite having 4× more Br₂ in Run C — that's your proof that Br₂ is zero order.

Part 2 — Finding n and m (Runs D and E)

Two more reactions using Run B as your baseline. Run D doubles the HCl. Run E doubles the acetone. Compare the rates to find the orders for each.

What You'll Measure

Bromine is orange. As it reacts, the solution goes colorless. You'll measure how fast the orange disappears using a spectrophotometer at 395 nm. The fading speed IS the reaction rate.

Heads up before you start

Run C will look much more orange than Run A — it has 4× more Br₂. Your first instinct will be that it should react faster. It won't. Same rate. That's the whole point of the experiment.

🎯 The Big Reveal — Why Scheme 1 Works

Three [Br₂] vs time lines — different starting heights, but identical slopes.

Same slope = same rate = zero order in Br₂.

The Bathtub Analogy

Imagine three bathtubs with the same drain. One has 1 inch of water, one has 2 inches, one has 4 inches. The drain rate (gallons per minute) is the same in all three — but the full tub takes longer to empty. The drain doesn't care how much water is in the tub. That's zero order.

In your lab: [Br₂] is the water level. k' is the drain rate. The drain (enolization step) doesn't involve Br₂ — so Br₂ level doesn't affect how fast it drains.

Once you've proven p = 0 from Runs A, B, C:

You don't need to test first-order or second-order plots. You're done with Br₂. Now you just compare Runs D and E against B to find n and m — same type of graph, same logic.

Why Is Br₂ Zero Order?

The surprising result has a real explanation. The reaction happens in two steps, and only one of them is slow.

1

Step 1 — Enolization (SLOW — controls the rate)

Acetone + H⁺ → enol form of acetone

CH₃COCH₃ + H⁺ → CH₂=C(OH)CH₃ + H⁺

This step requires acetone AND H⁺. Br₂ is NOT involved. This is the bottleneck — the factory floor that can only work so fast.

2

Step 2 — Bromination (FAST — not the bottleneck)

Enol + Br₂ → bromoacetone + HBr

CH₂=C(OH)CH₃ + Br₂ → CH₂BrCOCH₃ + HBr

This step IS so fast it's essentially instant once the enol forms. Having more Br₂ doesn't make this step faster — it's already waiting for enol to be produced.

The Factory Analogy

Step 1 is the factory making car bodies (slow). Step 2 is painting them (fast). The factory produces 10 bodies/hour regardless of how much paint you have. Bringing in more paint (= more Br₂) doesn't help — the bottleneck is making the bodies.

Zero order in Br₂ because Br₂ isn't at the bottleneck.

But Why Does [H⁺] Matter?

H⁺ IS at the bottleneck (Step 1). More H⁺ → faster enolization → faster overall reaction. That's why m = 1 (first order in H⁺). Same logic: more acetone → faster Step 1 → n = 1.

H⁺ is a catalyst. It helps make the enol in Step 1, then gets released in Step 2 and recycled. Net: H⁺ is not consumed overall. But its concentration still controls the rate of Step 1 — which is why it appears in the rate law.

Reading the Graphs

Part 1 — What to Look For

Runs A, B, C start at different heights (different [Br₂]₀) but fall with identical slopes. Three parallel lines. This is zero order.

The run with lowest [Br₂] (A) reaches zero first. The run with most Br₂ (C) takes longest. But the rate — the slope — is the same.

Part 2 — What to Look For

Run D has twice the H⁺ of Run B → slope is 2× steeper → m = 1.

Run E has twice the acetone of Run B → slope is 2× steeper → n = 1.

All three lines start at the same [Br₂]₀ (2×10⁻³ M) so comparing slopes is direct.

Procedure for Each Run

1

Set up

Get your 10 mL volumetric flask. Measure out the correct volumes before combining.

2

Add reagents in order

Br₂ solution first → HCl → acetone last. Adding acetone starts the reaction. Fill to 10 mL with DI water.

3

Mix and transfer immediately

Parafilm → invert 3× → into cuvette → into spectrophotometer. Start your stopwatch when acetone is added.

4

Record every 15 seconds

Continue until the solution goes colorless (absorbance ≈ 0) or 5–12 minutes (Run C takes longest).

⚠️ Critical tip: The biggest mistake is spending too long mixing and transferring. You'll miss the beginning of the reaction. Practice the sequence once before mixing reagents. Aim for: mixed → cuvette → reading, all within 30 seconds.

What to observe before starting each run:

  • Run A: pale yellow (low Br₂)
  • Run B: orange (2× more Br₂)
  • Run C: deep orange (4× more Br₂ — noticeably darker)

Prediction before Run C: Will it react faster, slower, or the same speed as Run A?

Write your prediction, then see if you're right.

Week 2 — Excel Analysis

Good news: You only need ONE type of graph — [Br₂] vs time — for all 5 runs. No comparing zero vs first vs second order plots.

Step 1: Standard Curve (Sheet 1)

1

2

Note your conversion equation

[Br₂] = (Absorbance − intercept) / slope

4

For each run's sheet

C2: =(A2 - intercept)/slope

Drag formula down. Column C = [Br₂] at each time point.

Step 3: Graph 1 — Runs A, B, C on same axes

5

Select and plot

On one chart: plot time vs [Br₂] for Runs A, B, and C as separate series. Add a trendline to each. Record the 3 slopes.

If slopes are equal → p = 0. Zero order in Br₂. Proven.

Step 4: Graph 2 — Runs B, D, E on same axes

6

Compare slopes

Plot time vs [Br₂] for Runs B, D, E. Add trendlines. Compare slopes:

slope_D / slope_B = ratio → = 2ᵐ → solve for m slope_E / slope_B = ratio → = 2ⁿ → solve for n

Step 5: Calculate k

k = k' / ([acetone] × [H⁺])   (when n=1, m=1)

Use the slope from Runs B, D, and E (they all share the same [Br₂]₀). Average your 3 k values.

Why not use Runs A and C for calculating k?

Runs A and C have different [Br₂]₀ than B, D, E — but since p=0, their slopes equal k×[acetone]×[H⁺] just the same. You CAN use them — but to keep comparisons clean, most instructors use B, D, E since they're designed as a matching set for finding n and m.
