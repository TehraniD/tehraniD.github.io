Objective

Determine the rate law for the acid-catalyzed bromination of acetone using a 5-run experimental design.

CH₃COCH₃ + Br₂ → CH₃COCH₂Br + HBr   (requires H⁺ catalyst)
rate = k [acetone]ⁿ [H⁺]ᵐ [Br₂]ᵖ   → Find n, m, p, and k

Experimental Strategy

Runs A, B, C (Part 1): Hold [HCl] and [acetone] constant. Vary [Br₂]. If the rate is unchanged → p = 0 (zero order in Br₂).

Runs D and E (Part 2): Use Run B as baseline. Double [H⁺] in Run D → find m. Double [acetone] in Run E → find n.
