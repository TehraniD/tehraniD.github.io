🔢 Part E: Equilibrium Calculations
Step-by-step procedure for each run:
1. Calculate [Fe³⁺]i = (2.00×10⁻³ M)(VFe / 10.0 mL)
2. Calculate [SCN⁻]i = (2.00×10⁻³ M)(VSCN / 10.0 mL)
3. Find [FeSCN²⁺]eq from absorbance using slope: [FeSCN²⁺]eq = Absorbance ÷ slope
4. [Fe³⁺]eq = [Fe³⁺]i − [FeSCN²⁺]eq
5. [SCN⁻]eq = [SCN⁻]i − [FeSCN²⁺]eq
6. Kc = [FeSCN²⁺]eq / ([Fe³⁺]eq × [SCN⁻]eq)

💡 Adding more Fe³⁺ increases [Fe³⁺], creating a stress on the system (Q < Kc). By Le Chatelier's principle, the equilibrium shifts to the right (forward direction) to consume the excess Fe³⁺ and produce more FeSCN²⁺, until a new equilibrium is established.
