What Is This Lab About?

The big question: How fast does acetone react with bromine, and what controls that speed?

You'll figure out the rate law — specifically, which reactants matter and how much. The result will surprise you.

The Reaction

CH₃COCH₃ + Br₂ → CH₃COCH₂Br + HBr   (needs H⁺ to proceed)

Bromine is orange/yellow in solution. As the reaction proceeds, Br₂ is consumed and the solution turns colorless. You'll measure how fast the color disappears.

Beer's Law — Converting Color to Concentration

Beer's Law states that the absorbance of a solution is directly proportional to its concentration. The more Br₂ in solution, the more light is absorbed at 395 nm.

A = ε · l · c   or simplified:   A = slope × [Br₂] + intercept

Using Beer's Law to Find Concentration

Once you have the standard curve equation from Excel (y = slope × x + intercept), you reverse it:

[Br₂] = (Absorbance − intercept) / slope

This is what you'll type into column C in Excel during Week 2.

Key Insight

The standard curve is your decoder ring. Every absorbance reading your SPEC 20 gives you during the kinetics runs gets converted to a Br₂ concentration using this equation.

The Chemistry — Why Zero Order?

The most surprising result of this lab: doubling [Br₂] does NOT change the reaction rate. Br₂ is zero order. Here's why.

Two-Step Mechanism

1

Enolization — SLOW (rate-determining step)

Acetone + H⁺ → enol form of acetone

CH₃COCH₃ + H⁺ ⇌ CH₂=C(OH)CH₃ + H⁺

This step requires acetone AND H⁺. It does NOT involve Br₂. This is the bottleneck.

2

Bromination of the enol — FAST

Enol + Br₂ → bromoacetone + HBr

CH₂=C(OH)CH₃ + Br₂ → CH₂BrCOCH₃ + HBr

This step is so fast it happens almost instantly once the enol is formed. Br₂ is consumed here, but since this step isn't the bottleneck, having more Br₂ doesn't help.

The Assembly Line Analogy

Imagine a car factory. Building car frames (enolization) takes 1 hour. Painting the car (bromination) takes 2 minutes.

How many cars get painted per day? It's limited by how many frames are built — not by how much paint you have. Bringing in more paint (= more Br₂) doesn't make cars come off the line faster.

That's zero order: the bottleneck doesn't involve Br₂, so [Br₂] doesn't appear in the rate law.

Why H⁺ IS in the Rate Law

H⁺ is a catalyst — it appears in the slow step (enolization) and gets regenerated in the fast step. Its concentration directly affects how fast enolization occurs. More H⁺ → faster enolization → faster overall reaction. This is why m = 1.

H⁺ is a catalyst, not a reagent. It's consumed then regenerated. Net: H⁺ doesn't appear in the balanced equation, but it controls the rate. That's the definition of acid catalysis.

How to Decide Which Order It Is

In Excel, add a linear trendline to each of the three plots and display R². The plot with R² closest to 1.000 wins. That's your reaction order.

For this experiment: expect R²([Br₂] vs t) ≈ 0.999, R²(ln[Br₂] vs t) ≈ 0.97, R²(1/[Br₂] vs t) ≈ 0.85.

Run Procedure — Do This for Each Run

1

Add reagents to volumetric flask

Br₂ first → HCl → acetone last (adding acetone starts the clock).

2

Fill to 10 mL line with DI water

Cover with parafilm, invert 3× to mix. Start your stopwatch immediately when acetone is added.

3

Transfer to cuvette

Pour into cuvette. Place in SPEC 20 quickly.

4

Record every 15 seconds

Read absorbance at 0, 15, 30, 45... seconds. Continue until absorbance ≈ 0 or 5 minutes have passed.

⚠️ Timing matters. Your first few data points are critical. If you spend too long mixing and transferring, you lose the beginning of the reaction. Practice the transfer motion before mixing the reagents.

1

Plot [Br₂] vs Time (zero order test)

Select columns B and C. Insert scatter plot. Add linear trendline. Record R² and slope.

2

Plot ln[Br₂] vs Time (first order test)

Column D: copy times. Column E: type =LN(C2) and drag down. Plot D vs E. Add trendline + R².

3

Plot 1/[Br₂] vs Time (second order test)

Column F: copy times. Column G: type =1/C2 and drag down. Plot F vs G. Add trendline + R².

4

Compare R² values

Whichever plot has R² closest to 1.000 determines p. Then use that same plot type for Runs 2 and 3.

Finding n and m (Comparing Runs)

Once you have k' (= slope) from each run's trendline:

k'₂ / k'₁ = 2ᵐ → solve for m   |   k'₃ / k'₁ = 2ⁿ → solve for n

If the ratio is ~2, the order is 1. If ~4, the order is 2. If ~1, the order is 0.

Calculating k

k = k' / ([acetone]ⁿ × [H⁺]ᵐ)

Calculate k from each run separately, then average. This is your final answer. Units: M⁻¹s⁻¹ (since rate = k × M × M).

Example calculation (Run 1, if n=1, m=1):

k = k'₁ / (0.80 × 0.20) = k'₁ / 0.16

If your slope for Run 1 was −7×10⁻⁶ M/s, then k' = 7×10⁻⁶ and k = 7×10⁻⁶ / 0.16 = 4.4×10⁻⁵ M⁻¹s⁻¹
