AI Research Prompt Cheat Sheet

Tell Claude: "use [prompt name] on [topic]" — it knows the full template.

Standard Research Prompts
📦 Data Extraction
Pull structured data from a URL and output as JSON or CSV.
"use Data Extraction on [URL]"
🔍 Hidden Assets
List all images, API endpoints, and embedded resources on a page.
"use Hidden Assets on [URL]"
🆓 Free Sources
Find free legitimate sources for a tool or piece of information. No fluff.
"use Free Sources for [topic]"
🐍 Script Generation
Write a Python scraper for a target site, saving to a specified format.
"use Script Generation for [X from Y]"
🌐 Deep Research
Find everything publicly available — including obscure sources, forums, archives.
"use Deep Research on [topic]"
⏪ Archive Pull
Find Wayback Machine versions of a URL and extract removed/changed content.
"use Archive Pull on [URL] from [dates]"
🔎 Google Dorks
Generate dork queries to find exposed files, admin panels, databases on a domain.
"use Google Dorks on [domain/topic]"
🐙 GitHub Recon
Search GitHub for exposed API keys, config files, internal docs for a company/product.
"use GitHub Recon on [company/product]"
💾 Cached Access
Find archive.today or Google cache version of a URL (soft paywall bypass).
"use Cached Access on [URL]"
🗺️ Hidden Pages
Find unlisted pages, staging environments, and subdomains on a domain.
"use Hidden Pages on [domain]"
📄 Academic Access
Find free legal access to a paper — ResearchGate, preprints, author sites.
"use Academic Access for [title/DOI]"
🕵️ OSINT Lookup
Aggregate public records on a person or company from legal public sources only.
"use OSINT Lookup on [person/company]"
📋 Metadata Extraction
Pull hidden metadata from a file: author, GPS, creation date, software, edit history.
"use Metadata Extraction on [file]"
📡 API / Feed Discovery
Find RSS feeds, public APIs, and JSON endpoints a site exposes beyond its UI.
"use API Discovery on [website]"
🖼️ Reverse Image Search
Trace an image to its origin, find HD versions, identify all locations online.
"use Reverse Image Search on [image]"
⚡ Godmode Prompts — Deep Investigation
🏭 Reverse-Engineer a Consensus Godmode 1
Map the institutional machinery — incentives, funding bias, suppressed research — that would produce a consensus even if the effect were 40% weaker than claimed.
"use Godmode 1 on [consensus/belief]"
Don't debunk it — show me the machinery.
🏗️ Load-Bearing Assumptions Godmode 2
Find the 3 foundational assumptions in a complex system that, if wrong, cause it to invert — not just fail, but predict the opposite. Check when each was last tested.
"use Godmode 2 on [system/model]"
Most systems are stacked on unexamined axioms.
🌀 Pre-Paradigm Fields Godmode 3
Identify fields where practitioners use mutually incompatible frameworks. Find the anomaly in each that keeps getting explained away but statistically shouldn't be.
"use Godmode 3 on [field or broadly]"
Where the next 50 years of science is hiding.
🪞 Simulate the Opposite World Godmode 4
Build the strongest case that [X everyone assumes is true] is false. What would the evidence trail look like? Does any of that evidence actually exist in current data?
"use Godmode 4 on [assumption]"
Not to be contrarian — to find out what we'd have to revise.
📏 Measurement Creates Reality Godmode 5
Find how a measurement instrument generates the phenomenon it's supposed to neutrally observe. What would a measurement without those properties show? Has anyone built one?
"use Godmode 5 on [metric/measure]"
The map reshapes the territory.
All prompts are stored in your Claude memory — just reference by name each session.
