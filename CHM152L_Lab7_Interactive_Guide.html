Key Concepts

What Is a Buffer?

A buffer is a solution that resists pH changes when small amounts of acid or base are added. It contains:

  • A weak acid (HA) — neutralizes added base (OH−)
  • Its conjugate base (A−) — neutralizes added acid (H+)

Both components must be present for a buffer to work!

How Buffers Work

When acid (H+) is added:

CH3COO− + H+ → CH3COOH

The conjugate base "absorbs" the added H+, converting to weak acid. pH barely changes.

When base (OH−) is added:

CH3COOH + OH− → CH3COO− + H2O

The weak acid "absorbs" the added OH−, converting to conjugate base. pH barely changes.

Henderson-Hasselbalch Equation

pH = pKa + log ([A−] / [HA])

This equation only works when both HA and A− are present (Beakers 2–5). Key insights:

  • When [A−] = [HA]: pH = pKa (the ratio is 1, log(1) = 0)
  • When [A−] > [HA]: pH > pKa (more basic)
  • When [A−] < [HA]: pH < pKa (more acidic)

Buffer Capacity

Buffer capacity is how much acid or base can be added before the pH changes significantly (> 0.5 units).

  • Against acid: depends on how much A− is available. More A− = more capacity against acid.
  • Against base: depends on how much HA is available. More HA = more capacity against base.
  • Maximum buffer capacity occurs when [HA] = [A−] (pH = pKa)

ICE Tables (for Beakers 1 & 6)

Beaker 1 (pure acid) and Beaker 6 (pure base) are NOT buffers — they lack one component. Use an ICE table instead.

Beaker 1 (weak acid): CH3COOH ⇌ H+ + CH3COO−

Use Ka = 1.8 × 10−5

Beaker 6 (weak base): CH3COO− + H2O ⇌ CH3COOH + OH−

Use Kb = Kw / Ka = 5.56 × 10−10

Buffers are critical in the human body! Blood is buffered at pH 7.35–7.45 by the carbonic acid/bicarbonate system (H2CO3/HCO3−). A pH drop below 7.35 (acidosis) or rise above 7.45 (alkalosis) can be life-threatening. IV fluids given to patients are buffered to prevent dangerous pH shifts.

Step-by-Step Procedure

Part I: Buffer Preparation

  1. Label six clean, dry 50 mL beakers 1–6.
  2. Using a graduated cylinder, prepare each solution:
    Beaker0.5 M CH3COOH (mL)0.5 M NaCH3COO (mL)Total (mL)Type
    110010Pure acid
    28210Buffer (4:1)
    36410Buffer (3:2)
    44610Buffer (2:3)
    52810Buffer (1:4)
    601010Pure base
  3. Measure and record the pH of each beaker using a calibrated pH meter.
  4. Rinse the probe with distilled water between each measurement.

Part II: Addition of Strong Acid (HCl)

  1. Add 5 drops of 3 M HCl to each beaker.
  2. Stir gently and record the new pH.
  3. Continue adding 5-drop portions, recording pH each time.
  4. Stop when the pH changes by more than 0.5 unit from the previous reading.
💡 The point where ΔpH > 0.5 means the buffer capacity has been exceeded.

Part III: Addition of Strong Base (NaOH)

  1. Prepare a fresh set of the six solutions (same as Part I).
  2. Add 5 drops of 3 M NaOH to each beaker.
  3. Continue adding 5-drop portions until ΔpH > 0.5.

💡 Tips for Success

  • Rinse the graduated cylinder with distilled water when switching between acid and base.
  • Wait for the pH meter reading to stabilize before recording (5–10 seconds).
  • Count drops carefully — consistency matters!
  • Don't forget to make fresh solutions for Part III.
